

VOLLEY:

add volley official repo as a module:
1. add volley sub-module:
    git submodule add -b master https://android.googlesource.com/platform/frameworks/volley volley
2. settings.grgadle, add volley module:
    include ':volley'
3. add compile dependency to app/build.gradle:
    compile project(':volley')
4. whenever volley changes, sync via:
    git submodule foreach git pull
